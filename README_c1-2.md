# Mission02. AI 브랜드 광고 영상 제작

## 브랜드: AI 개인비서 ‘똑똑이(TTOKTTOK AI)’

---

# 0. 제출 결과 요약

본 과제에서는 일상과 업무를 함께 관리해 주는 AI 개인비서 브랜드 **‘똑똑이(TTOKTTOK AI)’**를 주제로 10초 분량의 세로형 브랜드 광고 영상을 기획·제작하였다.

- **최종 영상:** `final_video.mp4`
- **영상 길이:** 10초 이내
- **화면 비율:** 9:16 세로형
- **권장 해상도:** 1080×1920
- **사용 도구:** ChatGPT 이미지 생성 또는 Midjourney, Kling AI 또는 Runway, ElevenLabs, Suno AI, CapCut
- **포함 요소:** AI 생성 인물 및 인터페이스, 홀로그램 AI 비서, 업무 처리 장면, 자막, 배경음악, 효과음
- **직접 촬영 여부:** 직접 촬영 없음
- **유료 스톡 사용 여부:** 유료 스톡 이미지 및 영상 사용 없음

본 광고는 바쁜 일상 속에서 여러 업무를 동시에 처리해야 하는 사용자가 AI 개인비서 ‘똑똑이’를 통해 문제를 해결하는 과정을 **문제 제시 → AI 등장 → 해결 → 브랜드 메시지 전달**의 구조로 표현하였다.

# 5. 프롬프트 수정 전·후 기록

## 5-1. 수정 사례 1: AI 비서 캐릭터 표현

### 수정 전 프롬프트

```text
A humanoid AI robot assistant standing next to an office worker.
```

### 문제점

- 로봇의 외형이 지나치게 기계적이고 차갑게 표현되었다.
- 인간형 얼굴이 부자연스럽게 생성되었다.
- AI 비서보다 공상과학 영화 속 로봇처럼 보였다.
- 친근하고 일상적인 브랜드 이미지와 충돌하였다.

### 수정 후 프롬프트

```text
A friendly futuristic AI personal assistant appearing as an elegant blue and cyan holographic light form, warm and trustworthy, minimal circular interface, no robot body, no exaggerated human face, premium technology commercial style
```

### 수정 이유

‘똑똑이’는 사람을 대체하는 로봇이 아니라 사용자의 옆에서 도움을 주는 디지털 비서이므로, 인간형 로봇 대신 추상적인 빛과 홀로그램 형태로 표현하였다.

### 수정 결과

- AI 캐릭터의 차가운 느낌이 감소하였다.
- 브랜드의 친근함과 전문성이 동시에 살아났다.
- 인물 얼굴 왜곡 문제도 줄어들었다.
- 다른 씬과의 색감 일관성이 향상되었다.

## 5-2. 수정 사례 2: 업무 장면의 복잡도

### 수정 전 프롬프트

```text
A person using many AI tools and working on multiple tasks.
```

### 문제점

- 화면에 너무 많은 아이콘과 텍스트가 무작위로 생성되었다.
- 주요 메시지가 무엇인지 알아보기 어려웠다.
- 인터페이스가 게임 화면처럼 복잡해졌다.
- 인물과 AI 비서가 화면 안에서 묻혔다.

### 수정 후 프롬프트

```text
A professional user with six clearly separated floating task panels: calendar, document writing, tax calculation, English learning, video planning and research, clean minimal holographic interface, blue and cyan color palette
```

### 수정 이유

업무 종류를 6개로 제한하고, 각각의 카드가 분리된 인터페이스로 보이도록 프롬프트를 구체화하였다.

### 수정 결과

- 업무 기능이 명확하게 전달되었다.
- 화면의 정보 밀도가 적절해졌다.
- 브랜드 컬러와 인터페이스 디자인이 통일되었다.

# 7. 기획 → 생성 → 검수 → 편집 파이프라인

## 7-1. 브랜드 기획

먼저 똑똑이의 브랜드 역할을 ‘질문에 답하는 챗봇’이 아닌 ‘사용자의 일과 삶을 연결하는 개인화 AI 비서’로 정의하였다.

핵심 키워드는 다음과 같이 설정하였다.

- 친근한 AI
- 정확한 업무 지원
- 통합형 개인비서
- 미래적 인터페이스
- 신뢰감
- 시간 절약

## 7-2. 스토리 구조 설계

10초 안에 메시지를 전달하기 위해 다음과 같은 4단계 구조를 사용하였다.

```text
업무 과부하
→ AI 비서 등장
→ 업무 해결
→ 브랜드와 슬로건 제시
```

각 장면은 독립적으로 예쁜 화면을 만드는 것보다, 앞 장면의 문제를 다음 장면이 해결하도록 연결하였다.

## 7-3. 이미지 생성

각 씬의 키 비주얼을 먼저 이미지 생성 AI에서 제작하였다.

공통 스타일 요소는 다음과 같이 고정하였다.

- 화면 비율: 9:16
- 색상: 딥 네이비, 블루, 시안
- 조명: 부드러운 홀로그램 글로우
- 분위기: 미래적이지만 차갑지 않은 프리미엄 테크 광고
- 인물: 동일한 성인 전문직 사용자
- 인터페이스: 원형과 카드형 UI
- 카메라: 시네마틱, 얕은 심도

## 7-4. 이미지 검수

다음 항목을 기준으로 생성 이미지를 검수하였다.

- 인물의 손이나 얼굴이 왜곡되지 않았는가
- 홀로그램 캐릭터가 지나치게 로봇처럼 보이지 않는가
- 화면의 UI가 너무 복잡하지 않은가
- 브랜드 컬러가 씬마다 동일한가
- 화면 중앙에 자막과 로고를 넣을 여백이 있는가
- 세로형 화면에서 주요 피사체가 잘리지 않는가

## 7-5. 영상 변환

검수한 키 비주얼만 Kling AI 또는 Runway에 입력하여 영상으로 변환하였다.

영상 생성 시에는 피사체의 큰 움직임보다 다음과 같은 미세한 움직임을 사용하였다.

- 카메라 줌인
- 홀로그램 빛의 파동
- UI 카드 정렬
- 작은 디지털 입자
- 체크 아이콘 등장
- 인물의 자연스러운 시선 변화

## 7-6. 음성 생성

내레이션은 짧고 명확하게 구성하였다.

### 내레이션 대본

전체 영상이 10초 이내이므로 실제 편집에서는 다음과 같이 압축한다.

```text
이제, 똑똑이에게 말하세요.
생각보다 가까운 AI,
당신만의 비서 똑똑이.
```

### 음성 톤

- 성별: 차분한 성인 남성 또는 여성
- 속도: 보통보다 약간 빠르게
- 감정: 신뢰감, 친근함
- 스타일: 광고 내레이션
- 과장된 감정 표현은 배제

# 8. 크레딧 및 대기열 제약 고려

## 8-1. 영상 생성 크레딧 절약

영상 생성은 이미지 생성보다 비용과 시간이 많이 소요되므로, 먼저 각 장면의 키 비주얼을 충분히 검수한 후 최종 이미지에만 모션을 적용한다.

Text-to-Video 방식은 장면의 구도와 캐릭터가 크게 변할 가능성이 있으므로 사용을 최소화하고, Image-to-Video 방식으로 제작한다.

## 8-2. 씬 수 최소화

10초 영상에 과도한 씬을 넣지 않고 4개의 장면으로 제한하였다.

각 장면은 2~2.5초 정도로 구성하고 메시지 전달에 불필요한 장면은 제외한다.

## 8-3. 인물 왜곡 대응

인물이 걷거나 손을 크게 움직이는 장면은 왜곡이 발생할 가능성이 높으므로 다음 움직임만 사용한다.

- 고개를 약간 드는 동작
- 화면을 바라보는 시선 이동
- 가벼운 호흡
- 카메라 줌인
- 홀로그램과 인터페이스 움직임

## 8-4. 대체 도구

# 11. 핵심 기술 원리 적용

## 11-1. T2I → I2V 단계적 통제

텍스트에서 바로 영상을 생성하면 인물, 인터페이스, 조명, 색감이 장면마다 달라질 수 있다.

따라서 다음의 단계적 제작 방식을 사용한다.

```text
텍스트 프롬프트
→ 이미지 생성
→ 이미지 검수
→ 이미지 기반 영상 변환
→ 편집 및 오디오 통합
```

이미지 단계에서 구도와 색감, 인물, AI 비서 형태를 먼저 확정한 후 영상 AI에서는 움직임만 추가한다.

이 방식은 브랜드 광고에서 중요한 시각적 일관성과 통제력을 높인다.

## 11-2. 물리적 표현을 활용한 프롬프트 설계

단순히 “친근한 AI”라고 작성하면 결과가 추상적이거나 캐릭터가 유아적으로 표현될 수 있다.

따라서 다음과 같이 구체적인 시각 요소를 지정한다.

- `soft blue holographic light`
- `warm and trustworthy presence`
- `no robot body`
- `no exaggerated human face`
- `minimal circular interface`
- `premium technology commercial`

감정이나 분위기를 추상적인 형용사만으로 전달하지 않고, 빛, 형태, 색상, 인터페이스와 같은 물리적 요소로 변환하여 프롬프트에 입력한다.

## 11-3. UI 텍스트 생성 오류 대응

이미지 생성 AI는 한글이나 세부 텍스트를 정확하게 생성하지 못하는 경우가 많다.

따라서 이미지 생성 단계에서는 화면 안의 텍스트를 최소화하고, 실제 자막과 로고는 CapCut에서 후반 작업으로 삽입한다.

# 12. 영상 및 오디오 요소

## 12-1. 시각 요소

- 홀로그램 AI 비서
- 일정, 문서, 세무, 영어, 영상 아이콘
- 디지털 알림창
- 체크 애니메이션
- 블루·시안 빛의 파동
- 미니멀한 브랜드 로고 화면

## 12-2. 청각 요소

- 미래적인 전자음
- 알림음
- UI 스와이프 효과음
- 체크 완료 효과음
- 차분한 광고 내레이션
- 신뢰감 있는 전자음악 BGM

## 12-3. 오디오 톤 통일

시각적으로 친근하고 전문적인 브랜드이므로 지나치게 무겁거나 공격적인 음악은 사용하지 않는다.

다크 사이버펑크 음악보다 따뜻한 전자음과 밝은 신스 사운드를 사용하며, 내레이션을 방해하지 않도록 BGM 볼륨을 낮게 조정한다.

# 13. 직접 촬영 및 유료 스톡 사용 여부

본 프로젝트는 이미지, 영상, 음성, 배경음악을 생성형 AI로 제작하였다.

- 직접 촬영 영상 사용 없음
- 유료 스톡 영상 사용 없음
- 유료 스톡 이미지 사용 없음
- 기존 광고 영상 재사용 없음
- 유명인의 얼굴 또는 음성 복제 없음
- 딥페이크 사용 없음

CapCut은 컷 편집, 자막, 색보정, 오디오 레벨 조정 등 통합 편집 용도로만 사용하였다.

# 15. 심층 인터뷰 답변

## Q1. 서로 다른 도구의 결과물을 통합할 때 발생한 불일치는 어떻게 보정했는가?

이미지 생성 AI에서 만든 키 비주얼은 선명하고 정적인 반면, 영상 변환 과정에서는 디테일이 약간 뭉개지고 색상이 어두워지는 문제가 발생할 수 있다.

이를 해결하기 위해 CapCut에서 모든 클립에 동일한 색보정 값을 적용하고, 약한 샤픈과 필름 그레인을 사용하여 질감을 통일하였다.

또한 이미지 생성 단계부터 딥 네이비, 블루, 시안 컬러를 공통으로 사용하여 도구가 달라도 브랜드 색감이 유지되도록 하였다.

## Q2. 같은 장면을 다른 영상 생성 도구로 만들면 어떤 차이가 예상되는가?

Kling AI는 인물과 피사체의 형태를 비교적 안정적으로 유지하면서 자연스러운 카메라 움직임을 만드는 데 유리하다.

Runway는 Motion Brush 등을 이용하여 화면의 특정 영역에 움직임을 주는 데 효과적이다.

Luma Dream Machine은 카메라 움직임이 역동적일 수 있지만, 홀로그램이나 UI 카드의 형태가 변형될 가능성이 있다.

본 광고는 화려한 움직임보다 AI 비서와 인터페이스의 형태 유지가 중요하므로, 보수적인 움직임을 적용할 수 있는 Kling 또는 Runway가 적합하다.

## Q3. 영상 생성 크레딧이 부족하다면 어떻게 제작할 것인가?

Midjourney 또는 ChatGPT 이미지 생성 결과물을 정지 이미지로 사용하고, CapCut 키프레임을 통해 확대, 이동, 패닝 효과를 적용한다.

여기에 홀로그램 오버레이, UI 효과음, 체크 애니메이션, 자막 전환을 추가하면 영상 생성 AI를 사용하지 않더라도 충분히 광고 영상처럼 보이게 만들 수 있다.

특히 Scene 4는 배경 이미지 한 장에 로고 애니메이션만 적용해도 완성할 수 있다.

## Q4. 10초보다 더 짧은 6초 광고로 줄인다면 어떻게 재구성할 것인가?

다음과 같이 3개 장면으로 축소한다.

```text
0~2초: 업무 과부하
2~4초: 똑똑이 등장과 업무 해결
4~6초: 로고와 슬로건
```

자막은 다음 두 문장만 사용한다.

```text
할 일은 많고, 시간은 부족하다.
생각보다 가까운 AI, 똑똑이.
```

## Q5. 내레이션을 사용하지 않고 광고를 제작한다면 어떻게 할 것인가?

내레이션 대신 자막과 효과음 중심으로 구성한다.

Scene 1에서는 여러 알림음이 겹치도록 하고, Scene 2에서 모든 소리가 순간적으로 멈춘 뒤 AI 등장 효과음을 삽입한다.

Scene 3에서는 각 업무 카드가 완료될 때마다 짧은 체크 효과음을 넣고, Scene 4에서 로고 리빌 사운드와 함께 슬로건을 보여준다.

## Q6. 똑똑이 브랜드가 다른 AI 서비스와 차별화되는 점은 무엇인가?

똑똑이는 단순한 정보 검색이나 문장 생성 기능만 제공하는 것이 아니라, 사용자의 업무와 일상 전반을 하나의 대화 안에서 지원하는 개인비서형 AI라는 점에서 차별화된다.

일정 관리, 문서 작성, 전문 업무, 콘텐츠 기획, 학습 등을 서로 분리된 기능으로 제공하는 것이 아니라 사용자의 목적을 중심으로 연결하여 지원한다.

# 17. 결론

본 프로젝트에서는 AI 개인비서 브랜드 ‘똑똑이’를 주제로 10초 분량의 세로형 브랜드 광고를 기획하였다.

광고는 현대인이 겪는 업무 과부하와 시간 부족 문제를 제시한 뒤, AI 개인비서 똑똑이가 등장하여 일정, 문서, 정보 검색, 전문 업무, 영어 학습, 콘텐츠 기획을 빠르게 정리하는 구조로 설계하였다.

시각적 일관성을 유지하기 위해 텍스트에서 직접 영상을 생성하는 방식보다, 먼저 이미지 생성 AI로 각 장면의 구도와 색감, AI 비서 캐릭터를 확정한 후 Image-to-Video 방식으로 움직임을 부여하는 제작 전략을 선택하였다.

또한 AI 비서가 지나치게 차갑거나 로봇처럼 보이지 않도록 인간형 로봇 대신 푸른 홀로그램 빛과 미니멀한 인터페이스를 사용하였다. 이를 통해 미래적이면서도 친근하고 신뢰할 수 있는 브랜드 이미지를 구현하였다.

최종적으로 이미지, 영상, 음성, 음악 등 서로 다른 AI 생성 결과물을 CapCut에서 통합하여 하나의 일관된 브랜드 경험으로 완성하였다.

본 과정을 통해 기획 의도, 프롬프트 설계, 결과 검수, 프롬프트 수정, 영상 변환, 오디오 생성, 최종 편집으로 이어지는 멀티모달 AI 콘텐츠 제작 파이프라인을 구체적으로 설계하고 적용할 수 있었다.
